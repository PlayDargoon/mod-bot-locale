# mod-force-locale - Принудительная локаль для ботов

## Описание

Модуль позволяет принудительно установить конкретную локаль для ботов AzerothCore, переопределяя стандартную систему приоритетов локалей.

## Как это работает

Стандартная система локализации ботов работает по принципу голосования - боты говорят на языке большинства подключенных игроков. Этот модуль переопределяет это поведение, устанавливая конкретную локаль из конфигурации.

При входе игрока модуль:
1. Сбрасывает все приоритеты локалей
2. Устанавливает настроенную локаль с высоким приоритетом (имитирует 100 игроков)
3. Гарантирует, что все боты используют принудительную локаль

## Требования

- AzerothCore с установленным mod-playerbots
- Этот модуль должен загружаться ПОСЛЕ mod-playerbots

## Установка

Модуль уже установлен в `azerothcore-wotlk/modules/mod-force-locale`

Пересоберите сервер:
```bash
./acore.sh compiler build
```

## Настройка

Отредактируйте `worldserver.conf` или создайте `force_locale.conf`:

```ini
# Включить модуль
ForceLocale.Enable = 1

# Установить локаль (0-8)
# 0 = enUS, 1 = koKR, 2 = frFR, 3 = deDE
# 4 = zhCN, 5 = zhTW, 6 = esES, 7 = esMX, 8 = ruRU
ForceLocale.Locale = 8
```

## Поддерживаемые локали

- 0 = enUS (Английский - США)
- 1 = koKR (Корейский)
- 2 = frFR (Французский)
- 3 = deDE (Немецкий)
- 4 = zhCN (Китайский - Китай)
- 5 = zhTW (Китайский - Тайвань)
- 6 = esES (Испанский - Испания)
- 7 = esMX (Испанский - Мексика)
- 8 = ruRU (Русский)
